# Leetcode: 706. Design HashMap.

**Ruby/Go: One huge array.**

Just create a huge array of 10 power 6 elements and save -1 as no value stored
by a key. A key is just an index in this array.

Ruby code:
```Ruby

class MyHashMap
    def initialize()
      @a = Array.new(10**6,-1)
    end


=begin
    :type key: Integer
    :type value: Integer
    :rtype: Void
=end
    def put(key, value)
      @a[key] = value
        
    end


=begin
    :type key: Integer
    :rtype: Integer
=end
    def get(key)
      @a[key]
    end


=begin
    :type key: Integer
    :rtype: Void
=end
    def remove(key)
      @a[key] = -1
    end


end
```
