# Leetcode: 413. Arithmetic Slices.

Small tasks.
```Python
def solve(st):
    x = list(st)
    for i in range(0,len(x)//2):
        j = len(x) - 1 - i
        a = ord(x[i])
        b = ord(x[j])
        if a + 1 == b - 1: continue
        if a - 1 == b + 1: continue
        if a == b: continue
        return False
    return True;
```

```Ruby
def solve st
  st.chars.zip(st.chars.reverse).all?{|a,b| a == b || 2 == (a.ord - b.ord).abs}
end
```
