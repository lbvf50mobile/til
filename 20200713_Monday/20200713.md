Bash Brace Expansion do not use spaces to make it works.
- `echo a{a, b}c # a{a, b}c`
- `echo a{a,b}c # aac abc`

```Shell
#!/usr/bin/env bash
for f in *; do for x in $f/*; do mv $x ${x:0:1}_${x:2} ; done ; done
# does not copy all files:
# cp -R {1,2,3,4,5,6,7,8,9}/. all2/
```
